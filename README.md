# mary-online-store
Online Store Full Stack (Java 8, Spring Boot, AngularJS, Bootstrap)

**Before  you start:**
1. install git
2. install jdk8
3. install maven2


**How to start:**
1. ```git clone https://github.com/alexanderkarpov/mary-online-store.git```
2. ```cd mary-online-store```
3. ```mvn spring-boot:run```
4. visit ```localhost:8080```
